## 📌 Overview

**Smart RoadSOS** is a location-based emergency assistance web application designed to provide instant access to nearby critical services during road accidents.

In emergency situations, every second counts. This system helps users quickly locate hospitals, police stations, ambulance services, and vehicle support—reducing delays during the **Golden Hour** and improving chances of survival.

---

## 🌟 Key Features

- 📍 **Live Location Detection**  
  Detects user location using browser geolocation  

- 🚨 **One-Tap Emergency Alert**  
  Generates alert message with live location link  

- 🏥 **Nearby Emergency Services**  
  - Hospitals  
  - Police stations  
  - Ambulance services  
  - Towing / repair shops  
  - Puncture services  

- ⚡ **Golden Hour Mode**  
  Prioritizes nearest hospitals for faster response  

- 📶 **Offline Mode**  
  Displays cached data in low/no network conditions  

- 🎤 **Voice Activation**  
  Trigger emergency actions using voice commands  

- 🧠 **AI Emergency Guide**  
  Provides step-by-step guidance during accidents  

- 🌍 **Global Applicability**  
  Works across regions using map APIs
